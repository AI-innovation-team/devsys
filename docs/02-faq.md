# 常见问题

## 我看不到某台服务器 / 按钮是灰的？

说明你还没为它配置凭据。点**头像 → 设置**填写用户名与密钥/密码即可。

## 工作区和普通 SSH 有什么区别？

- **普通 SSH**：临时终端，关掉网页会话就结束。
- **工作区**：`tmux` 持久会话，关掉网页只是断开，进程照跑，回来可继续接入。

## 关掉浏览器，我的训练会中断吗？

- 在**工作区**里跑：**不会**，回来点「打开」继续。
- 在普通 **SSH** 里跑：**会**，请改用工作区。

## 我的凭据安全吗？

凭据经加密后存于门户，仅用于以你的身份连接目标机。门户不共享、不持有 root 权限。
