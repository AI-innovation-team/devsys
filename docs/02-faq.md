# 常见问题

## 怎么登录？

**申请制**，两种方式：**GitHub** 一键登录，或**邮箱 + 密码**。需要账号请联系管理员。

## 邮箱账号的初始密码是什么？

`modifyme2026`。登录后请到 **设置 → 账户** 改成自己的密码。

> 密码请用纯字母数字，避免在中文输入法下打出全角符号（如 `（！）`）导致登录失败。

## 我改不了密码 / 看不到「账户」？

- 改密码只对**邮箱登录**的账号开放（GitHub 登录没有密码）。
- 邮箱账号登录后：**设置 → 账户** → 填新密码 → 修改。

## 我看不到某台服务器 / 按钮是「设置凭据」？

说明你还没为它配置凭据。点**头像 → 设置**填写用户名与密钥/密码即可。

## 工作区和普通 SSH 有什么区别？

- **普通 SSH**：临时终端，关掉网页会话就结束。
- **工作区**：`tmux` 持久会话，关掉网页只是断开，进程照跑，回来点「打开」继续接入。

## 关掉浏览器，我的训练会中断吗？

- 在**工作区**里跑：**不会**，回来点「打开」继续。
- 在普通 **SSH** 里跑：**会**，请改用工作区。

## 我的凭据安全吗？

服务器连接凭据经加密后存于门户，仅用于以你的身份连接目标机。门户不共享、不持有 root 权限。
