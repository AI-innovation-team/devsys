# Web SSH 终端

在**服务器**页点某台机器的 **SSH**,即在浏览器里开一个终端,以你自己的 Linux 账号连接。

## 何时用

- 快速看一眼日志、改个配置、跑个短命令。
- 不需要长时间运行、关掉网页也无所谓的操作。

## 特性

- 全功能 xterm 终端,支持颜色、光标、常见按键。
- 已内置 Nerd Font 字形,`starship`、`lsd` 等的图标能正常显示。
- 连接凭据经 Fernet 加密存于门户,仅用于以你的身份发起连接。

## 注意

Web SSH 是**临时会话**:关闭网页或断网,这个终端里的前台进程会随之结束。
需要长时间运行的任务,请改用[持久工作区](03-workspaces)。
